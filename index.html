Schema Validation Utilities

Provides support for validating XML representations using the following schema languages:

  • W3C XML Schema 1.0
  • Schematron (ISO/IEC 19757-3:2006)
  • RELAX NG (ISO/IEC 19757-2:2008)

It is also possible to access and query the post-schema-validation infoset (PSVI) using the Xerces XML Schema API.
