Release Notes

1.6 (2015-04-28)

This release includes no functional changes. Several dependencies and Maven plugins have been updated. Accessing a special repository is no longer required.

1.5 (2014-05-20)

The project is now hosted at GitHub. This release introduces new site content, but the essential functionality of the library is unchanged.

  • Modify POM for GitHub.
  • Add new site content.
  • Change license to Apache License, Version 2.0.

1.4 (2014-04-15)

This minor release includes the following updates:

  • Fix Javadoc errors reported when building with JDK 8.
  • Set system identifier for retrieved XML Schema source.
  • Update dependencies to latest releases.
